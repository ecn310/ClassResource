7.1 LaTeX Overview

LaTeX is like a fancier version of rmarkdown. You can explore its basic commands on the LearnLaTeX website.

– You have the source file where you write the “code” and then you compile/typeset it into the final product - Final product is usually PDF - We will use www.overleaf.com - Structure: - Preamble (set options for the document) - \begin{document} - Body of document - \end{document}

Rules

  • Many commands need an opening and a closing
  • \begin paired with \end
  • Then { paired with }. Example: – $ to start math paired with another $ to show where the math ends • Others just show up alone – \section – –
